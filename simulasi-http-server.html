Simulasi Project 02

Simple HTTP Server dengan Python

Simulasi ini menggambarkan cara menjalankan website lokal menggunakan Python HTTP Server, lalu membukanya melalui alamat localhost.

01

Siapkan folder website

File HTML, CSS, dan JavaScript berada dalam satu folder project.

02

Jalankan server lokal

Perintah Python digunakan untuk membuat server sederhana.

03

Buka alamat localhost

Browser menampilkan halaman dari server lokal tersebut.

LOCAL SERVER DEMO

Terminal

$ cd website-sederhana
$ dir
index.html style.css script.js
$ python -m http.server 8000
Serving HTTP on 0.0.0.0 port 8000...
127.0.0.1 - - "GET /index.html HTTP/1.1" 200

Browser

http://localhost:8000/index.html

Website berhasil tampil

Halaman lokal dapat dibuka karena file project dilayani oleh server Python.
