Simulasi Project 03

Konfigurasi MikroTik Dasar

Simulasi ini menggambarkan proses konfigurasi RouterOS dasar untuk membuat jaringan LAN, DHCP, NAT internet, dan pengujian koneksi.

01

Atur IP LAN

Menambahkan alamat gateway pada interface bridge-LAN.

02

Aktifkan DHCP

Membagikan IP otomatis untuk perangkat client di jaringan lokal.

03

Tambahkan NAT

Mengaktifkan masquerade supaya client dapat mengakses internet.

04

Uji Ping

Memastikan router dan client dapat terhubung dengan baik.

ROUTEROS DEMO

RouterOS Terminal

/ip address add address=192.168.10.1/24 interface=bridge-LAN
address added: 192.168.10.1/24
/ip dhcp-server setup interface=bridge-LAN
dhcp pool: 192.168.10.10-192.168.10.100
/ip firewall nat add chain=srcnat out-interface=ether1 action=masquerade
nat rule active: masquerade
/ping 8.8.8.8 count=3
sent=3 received=3 packet-loss=0%
